# dotfiles

Layout:
- Dot-prefixed files (`.bashrc`, `.gitconfig`, ...) symlink into `~/`
- Other directories (`i3/`, `nvim/`, ...) symlink into `~/.config/`

## Software
- Terminal: Ghostty
- Text Editor: Nvim
- Window Managers: i3 (X11), sway (Wayland)
- Bar: waybar
- Notifications: mako
- Launcher: fuzzel
- Multiplexer: tmux
- Shell: bash
